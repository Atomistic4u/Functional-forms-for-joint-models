# Functional forms for joint models for longitudinal and time-to-event data

This repository accompanies the paper

> Functional forms in joint models for longitudinal and time-to-event data:
> A practical guide with application and interpretation.

It contains example R code illustrating how to specify the different
functional forms implemented in the JMbayes2 package using the
publicly available Primary Biliary Cirrhosis (PBC) dataset.

## Files

- `functional_forms_pbc.Rmd` — R Markdown illustrating the syntax.
- `functional_forms_pbc.html` — Rendered HTML version.
